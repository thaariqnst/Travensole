# **Travensole - Your Smart Travel Companion**  

### **Overview**

**Travensole** is an innovative travel management software designed to cater **Travelers**

It's built with Java **Object Oriented principles**, this application ensures any reservation made is taken. Each query is provided with proficient information regarding the reservation, whether it's Flights or Hotels.

Travensole's README and interface is highly inspired by [Travelogix](https://github.com/AfrinJahanEra/Travelogix) but with simpler features. Check them out here:
